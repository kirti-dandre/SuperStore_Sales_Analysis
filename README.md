# SuperStore_Sales_Analysis

Analyzed Superstore sales data using SQL to identify sales trends, customer behavior, and top-performing products. The project demonstrates the use of SQL queries for business intelligence and data-driven decision-making.

#Tools Used
-SQL
-MySQL Workbench
-MySQL Database
#Analysis Performed
-Top States by Sales
-Sales by Category
-Sales by Customer Segment
-Top Customers by Spending
-Top Products by Revenue
#SQL Concepts Used
-SELECT
-GROUP BY
-ORDER BY
-SUM()
-COUNT()
-LIMIT
#Key Insights
California generated the highest sales.
Consumer segment contributed the most sales.
A small number of customers generated significant revenue.
Certain products consistently outperformed others.
